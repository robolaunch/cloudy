
 ## Hardware Setup

This page details the hardware setup instructions for Cloudy.

## Tools Needed

 - Philips screwdriver
    For Screwing screws
 - Pliers
    For Removing supports
 - Circlip pliers
    for securing circlips
 - Assembly hammer
    To tightly fit bearings

## Step 2

 Assemble M4 hex nuts as shown in the picture below.

You will need;
- pliers for inserting nuts
- Hammer for tapping nuts

<img src="https://raw.githubusercontent.com/robolaunch/cloudy_stl/main/images/CASTERWHEELS%20NUTS.png" width="800" height="430" align="top">


------------------------------------------------------------------------------
## Step 3

 Fit skf-634 bearings in the pockets as shown in the picture below.

 Lightly tap on outer rings of the bearings. bearings should fit tightly.

 Use circlip pliers and circlips to secure bearings in their places, note that some bearings dont have circlip grooves those bearings will be secured when assembled to the frame 


You will need;
- Hammer for bearing assembly
- circlip pliers 

<img src="https://raw.githubusercontent.com/robolaunch/cloudy_stl/main/images/CASTERWHEELS%20BEARINGS.png" width="800" height="430" align="top">

------------------------------------------------------------------------------
