
 ## Hardware Setup

This page details the hardware setup instructions for Cloudy.

## Tools Needed

 - Philips screwdriver
    For Screwing screws
 - Pliers
    For Removing supports
 - Circlip pliers
    for securing circlips
 - Assembly hammer
    To tightly fit bearings

## Mechanical assembly

This 17 step assembly guide will guide you through complete mechanical assembly.
